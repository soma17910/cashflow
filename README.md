# キャッシュフロー表メーカー

個人・個人事業主・法人向けの、とても見やすいキャッシュフロー表アプリです。

**公開URL**: https://soma17910.github.io/cashflow/

## できること
- **3つの区分**（個人 / 個人事業主 / 法人）で入力項目とアウトプットが自動で切り替わります
- これからの **12か月の収支** を月次表・グラフで見える化
- 突発的な支出に備えて **最低いくら手元に残すべきか（生活防衛資金）** を自動算出
- 収入減・突発出費の **ストレステスト**（もしもシミュレーション）
- 個人事業主の税・国保・年金、法人の法人税・消費税積立を概算で自動計算
- 円／万円の単位切替、入力の保存、印刷・PDF保存

## iPhone / スマホで使う
1. Safari で https://soma17910.github.io/cashflow/ を開く
2. 共有ボタン → **「ホーム画面に追加」** を選ぶ
3. アイコンから、アプリのように全画面で使えます

## 技術
- 単一HTMLファイル（外部ライブラリなし・完全オフライン動作）
- データはブラウザ内のみに保存され、外部に送信されません

> 税額・保険料は簡易的な概算です。正確な数値は税理士・社労士等の専門家にご確認ください。
